About This Project
Problem

Over time, computers accumulate duplicate files — the same photos, documents, music, and downloads scattered across different folders. These duplicates waste valuable disk space and make file management harder.

Solution

Smart File Duplicate Detector scans any folder on your computer, computes SHA-256 hashes for every file, and identifies exact duplicates. It groups duplicate files together, shows you how much space can be saved, and lets you delete extra copies with a single click.

How It Works
  1. Select a folder — Enter any folder path on your computer.
  2. Hashing — Each file is read in 64 KB chunks and a SHA-256 hash is computed.
  3. Compare — Files with identical hashes are exact duplicates.
  4. Review — Duplicate groups are displayed with file names, sizes, and locations.
  5. Clean up — Delete unwanted duplicates and free up disk space.
